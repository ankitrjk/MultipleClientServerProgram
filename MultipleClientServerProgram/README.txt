First run the all the hosts file, then server4.py and then client4.py:
$python3 HostA.py
$python3 HostB.py
$python3 HostC.py
$python3 HostD.py
$python3 server1.py
$python3 server2.py
$python3 client.py

Now enter the username and password when prompted.
Enter the frame size in KB and probability of corrupting the frames.
Enter 'EXIT' for closing the connection with server.

IP addresses and Port no of hosts, servers and clients can be changed by changing the entries in HostName.rtl file.
Make sure no Port no are repeated.
